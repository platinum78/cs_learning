# SWExpertAcademy #7333

Deque를 사용하여 문제를 해결하였다.
상자의 무게를 오름차순으로 정렬한 뒤, 가장 무거운 박스를 하나 빼고, 이 박스의 무게 * 총 박스의 개수 가 50kg을 초과할 때까지 가벼운 상자들을 하나씩 뺀다.
최종적으로, 가벼운 쪽 상자의 인덱스가 무거운 쪽 상자의 인덱스를 초과하면 모든 박스를 다 옮긴 것이므로, 프로그램을 끝내고 스택의 갯수를 반환한다.
이 때, 마지막 스택은 50kg을 채 채우지 못했을 가능성이 높으므로, 이 스택의 무게를 검사한 뒤, 만약 무게를 체웠다면 그대로 두고, 채우지 못했다면 스택 갯수를 하나 감소시켜 반환한다.
