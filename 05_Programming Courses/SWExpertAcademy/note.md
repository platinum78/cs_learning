## Greedy 알고리즘

### Knapsack 문제: 물건의 무게와 가격이 주어질 때, 무게를 초과하지 않으면서 가격을 최대로 할 수 있는 방법을 찾는 문제.
* 물건을 쪼갤 수 없는 경우: 완전 검색이 답.
* 물건을 쪼갤 수 있는 경우: Greedy 알고리즘으로 해결 가능.

### 회의실 배정 문제: 시작과 종료 시간이 있는 회의들이 주어질 때, 한 개의 회의실에서 가장 많은 수의 회의가 열릴 수 있도록 하는 방법
* 종료 시간 순서대로 정렬하고, 회의 시간이 겹치지 않는 것들 중 종료 시간이 가장 빠른 회의를 선택해 배정한다.

### Baby Gin 문제
* 먼저 Run이 되는지 확인하고, 해당 Index의 값들을 1씩 차감한 다음, Triplet 여부를 다시 확인한다.
